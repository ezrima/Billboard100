Software Required:
MySQL Workbench | https://dev.mysql.com/downloads/mysql/
MySQL Community Server | https://dev.mysql.com/downloads/mysql/
Java Runtime Environment (Latest) | https://www.java.com/en/download/

Running The Software:
In MySQL Workbench, open the script file “billboardnumberones.sql” and execute the script. This will create the database and populate it with data.
Set “billboardnumberones” as your default schema.
Ensure that MySQL Community Server is running on your computer using the Server menu.
Open Windows Command Prompt and enter the following commands:
“cd <directory containing the BillboardNumberOnes.jar file>
“java -jar BillboardNumberOnes.jar”
The program will be initiated in the command line.

Source Code:
The source code is included in BillboardNumberOnes.java and can be viewed in any IDE that supports Java.
